
Two-dimensional TLM in Java

1. To learn how to use the 2d-TLM applet, read the file "index.html" with you html-browser.


2. There are two possibilities to run the 2d-TLM program:

2.1. Use your java-enabled html-browser or the jdk-appletviewer to open the file "zweidtlm.html".
     The applet cannot save any geometry data and simulation results to files.

2.2. Start the applet with the java virtual-machine. The command may look like this:
     "java -jar zweidtlm.jar" or "jview /cp zweidtlm.jar zweidtlm".
     In this mode, the program can access the file-system for storing data and simulation results.


This archive "zweidtlm.zip" contains 18 files.
